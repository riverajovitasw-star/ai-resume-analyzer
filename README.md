# AI Resume Analyzer

## Overview

AI Resume Analyzer is a full-stack web application designed to help job seekers evaluate and improve their resumes. The system analyzes uploaded resumes, compares them with a given job description, and provides valuable insights such as ATS score, job match percentage, matched skills, missing skills, and improvement suggestions.

The project aims to simplify the resume screening process and help users understand how well their resumes align with industry requirements.

---

## Features

* Resume upload and analysis
* ATS (Applicant Tracking System) score calculation
* Job description matching
* Skill extraction from resumes
* Missing skill identification
* Resume improvement suggestions
* Performance dashboard with graphical representation
* Downloadable improved resume report
* Modern and responsive user interface

## Project Workflow

1. Upload a resume file.
2. Enter or paste a job description.
3. Analyze the resume.
4. View ATS score and job match percentage.
5. Review matched and missing skills.
6. Get resume improvement suggestions.
7. Download the improved resume report.
